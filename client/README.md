# Kashmir Tourism React Frontend

Modern React frontend with AI predictions and resource planning.

## Setup

```bash
cd client
npm install
```

## Environment

Copy `.env.example` to `.env`:
```bash
cp .env.example .env
```

## Run

```bash
npm start
```

App will run on http://localhost:3000
